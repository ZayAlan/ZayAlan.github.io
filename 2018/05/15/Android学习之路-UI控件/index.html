Android学习之路--UI控件

发表于 2018-05-15

以下是个人使用不熟练的控件属性记录，更多请参考developer文档

UI控件 属性
TextView android:gravity 指定文字的对齐方式，可选top,bottom,left,right,center等
Button android:textAllCaps 大小写设置
EditText android:maxLines 指定最大行数
ImageView 在Java文件中可通过setImageResource更改图片源
ProgressBar 在Java文件中可通过setVisibility进行可见的设定，分别有View.VISIBLE、View.INVISIBLE和View.GONE这三种值；在xml中可以通过style="?android:attr/progressBarStyleHorizontal"给进度条指定不同的样式，再使用android:max设定最大值，最后在Java文件中用.getProgress()和.setProgress(int)方法对其进行进度条进度的更改

AlertDialog控件

在当前界面弹出一个至于所有界面元素之上的对话框。

ProgressDialog

与AlertDialog不同的是，ProgressDialog会在对话框中显示一个进度条。
