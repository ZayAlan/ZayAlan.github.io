软技能-学习方法

发表于 2018-05-25

《软技能:代码之外的生存之道》一书中，作者谈到为了掌握一门技术，需要了解以下三个要点：

  • 如何开始——要想开始使用自己所学的，我需要掌握哪些基本知识？

  • 学科范围——我现在学的东西有多宏大？我应该怎么做？如果我能够对该学科的轮廓有大致的了解，那么将来我就能发现更多的细节。

  • 基础知识——要想使用一项特定的技术，我需要了解基本的用户案例和最常见的问题，也需要了解自己学习哪些东西就可以满足绝大多数情况下的日常应用。

“十步学习法”

学习开始前

  1. 了解全局。了解自己将要学习的主题的全局，对自己想要学的课题做一些基础性研究。

  2. 确定范围。在学习过程中，不要试图解决太大的问题，需要运用自己在上一步中获得的信息，让自己的关注点落脚到更小也更可控的范围。把大主题分解成可控的小主题。注意明确考虑时间因素。

  3. 定义目标。明确自己学习东西成功之后，需要达到的目标，用以评估自己是否已经达成学习目标，促使自己向着既定目标不断前进。

  4. 寻找资源。尽可能多的寻找与所选主题相关的资源。

  5. 创建学习计划。根据找到的资源概览，创建自己的学习路径，打造好自己的学习计划。

  6. 筛选资源。决定使用哪些资源来完成自己的学习任务。

开始学习，循环往复

学习——实践——掌握——教授

  1. 开始学习，浅尝辄止。获得足够多的与所学主题相关的信息，以便下一步中动手操作。要注意专注于掌握自己所需的、能在下一步动手操作的最小量的知识。

  2. 动手操作，边玩边学。将上一步学习的信息付诸于实践操作，并总结操作过程中出现的问题和遗忘的点，记录下来。

  3. 全面掌握，学以致用。在上一步发现不足提出问题的基础上，利用资料解决问题，进行深入学习，推动终极目标前进。

  4. 乐为人师，融会贯通。将所学进行精简，并将之以自己的语言记录或讲解给别人听，使他人能通过你而学习理解。
